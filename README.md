# personalPokedex
Pokedex website created to display pokemon

## Tools Used:
* HTML
* CSS
* JS
* Materialize
* Axios


## Features:
* Calls the Pokemon API using Axios
* CSS/JS Animations
* Mobile Responsive with Materalize
